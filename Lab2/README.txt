	Для успешного запуска программного модуля следует:
1. Установить интерпретатор Python и библиотеку matplotlib.
2. Поместить входной файл nfcapd_20200.csv и саму программу в одну директорию.
3. Запустить модуль из командной строки после перехода 
в данную директории путем вызова ввода команды "python labtwo.py".
4. Вывод работы программы будет осуществлен в командную строку, график появится в отдельном окне.
