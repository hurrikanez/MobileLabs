	Для успешного запуска программного модуля следует:
1.Установить интерпретатор Python c официального сайта и добавить его вызов в PATH.
2. Поместить входной файл CSV и саму программу в одну директорию.
3. Запустить модуль из командной строки после перехода 
в данную директории путем вызова ввода команды "python labone.py".
4. Вывод работы программы будет осуществлен в командную строку.
